# Domain Glossary

Shared vocabulary for Invoice Maker. Use these terms consistently in code, docs, and UI copy.

| Term | Definition |
| --- | --- |
| **Organization** | Tenant account; owns clients, invoices, and settings |
| **Client** | External party billed by an organization |
| **Invoice** | Bill document with line items, totals, and lifecycle status |
| **LineItem** | Single row on an invoice (description, qty, price, tax) |
| **Payment** | Money received against an invoice (full or partial) |
| **Draft** | Invoice being edited, not yet sent |
| **Sent** | Invoice delivered to client, awaiting payment |
| **Paid** | Invoice fully settled |
| **Overdue** | Sent invoice past due date without full payment |
| **Void** | Cancelled invoice; no longer collectible |
